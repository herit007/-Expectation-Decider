# 📊 Probability & Statistics Project

## 📌 Overview

This project demonstrates the application of probability concepts using a student performance dataset. It focuses on analyzing relationships between student activities and exam outcomes using statistical methods.

---

## 📁 Project Structure

* `Expectation_Decider.ipynb` → Main notebook containing practical implementation (Q3–Q7)
* `student_probability_dataset_300.csv` → Dataset used for analysis
* `Probability_Theory_Answers.pdf` → Theory answers for Q1, Q2, and Q6

---

## 🎯 Objectives

* Understand and apply probability concepts
* Construct probability distributions
* Analyze relationships between events
* Apply conditional probability and Bayes Theorem

---

## 📂 Dataset Description

The dataset contains information about students, including:

* Study Hours
* Attendance Percentage
* Group Discussion Participation (Yes/No)
* Previous Test Scores
* Final Exam Result (Pass/Fail)

---

## 🧠 Concepts Implemented

### 🔹 Random Variable & Probability Distribution

Defined a random variable:

> X = Number of students passing out of 3 randomly selected students

Constructed probability distribution using binomial expansion.

---

### 🔹 Venn Diagram

Visualized the relationship between:

* Study Hours (>10 hours)
* Attendance (>80%)

---

### 🔹 Joint, Marginal & Conditional Probability

* Created contingency table
* Calculated joint probability
* Computed marginal probabilities
* Derived conditional probabilities

---

### 🔹 Independence of Events

Checked whether events are independent using:

> P(A ∩ B) = P(A) × P(B)

Conclusion: Events are dependent.

---

### 🔹 Bayes Theorem

Applied Bayes Theorem to compute:

> P(Discussion | Pass)
